# Contributing Guidelines

Thank you for considering contributing to this project!

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Commit your changes with clear messages.
4. Submit a pull request for review.

## Code Style
- Follow clean, readable, and well-commented code practices.
- Keep commits small and focused on one change at a time.

## Reporting Issues
If you find a bug or issue, please open an issue with details and steps to reproduce the problem.
